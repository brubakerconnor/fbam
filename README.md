
# fbam

[![DOI](https://zenodo.org/badge/823447177.svg)](https://zenodo.org/doi/10.5281/zenodo.12631845)

Frequency Band Analysis for Multiple Stationary Time Series

## Overview

This package implements a genetic algorithm for the optimization of the
FBAM objective function to perform frequency band analysis of multiple
stationary time series.

## Installation

You can install `fbam` using the `devtools` package.

``` r
devtools::install_github("brubakerconnor/fbam")
```
